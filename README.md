# Tudor's Billionth Second

A tiny static website for Tudor's billion-second birthday.

## Pages

- `index.html` — the real timer
- `debug.html` — a test page for the final countdown and celebration effects

## GitHub Pages

The repository includes a GitHub Actions workflow for deploying this static site to GitHub Pages.

Expected site URL once Pages is enabled/deployed:

`https://andreineculai.github.io/tudor-timer/`

If GitHub asks for a Pages source, choose **GitHub Actions**.
